

A. UX Designer
   Persiapan yang diperlukan seorang UX design untuk meningkatkan kualitas produk digital :

1. menentukan jenis website yang akan kita buat(contoh website marketplace)

2. Competitive research -> mencari kekurangan dan kelebihan website marketplace lain, lalu di tulis dalam dokument.

3. Ux Survey -> contoh menggunakan google form google.com/forms yang berisi pertanyaan - pertanyaan seputar marketplace.

4. Customer persona (membuat profile untuk calon pengguna)
   -> profile nantinya digunakan untuk mencari audient yang bersedia di interview untuk melakukan usebility testing

5. User flow & journey -> digunakan untuk membuat paduan pengguna website kita
   - (https://www.mindmeister.com)

6. Prototyping dan User-testing -> Wireframe lebih mirip dengan architectural blueprints, sedangkan prototype merupakan representasi        high fidelity dari produk yang sudah final.




B. Figma
Figma adalah Aplikasi desain UI dan UX editor grafis vektor dan alat prototyping dengan berbasis web

1. create team(new project) -> new file -> rename untitled 

2. Frame/akboard (kerangka dasar) -> new frame (rename frame) (shortcut - F)

3. Layer -> struktur element, untuk Posisi komponen di layer (posisi komponen diatas, maka posisi komponen berada di tumpukan teratas) 

4. assets -> tempat untuk menyimpan komponen yang bisa digunakan berulang

5. pages/halaman -> folder

6. group -> shift & select -> klik kanan(group selection)

   - ctrl & d (duplikat)

7. Rectangle/container -> biasa digunakan untuk background

8. Teks tools -> alt & arah(menggeser), alt & klik(duplikat) 

9. image -> ctrl & klik arah mouse (crop)

10. shapes/bentuk/teks

11. Button UI -> buat dengan Rectangle


12. Prototype adalah fitur yang memudahkan kamu untuk menguji rancangan desain yang sudah kamu buat dengan mencoba menjalankannya seperti sebuah aplikasi. Fitur ini berguna ketika kamu ingin membuat beberapa rancangan tampilan secara realâ€“time dan nyata.

    Di dalam fitur ini, kamu akan merasakan bagaimana rancangan dasar dari aplikasi yang sedang kamu buat. Prototype ini memberikan pengalaman rancangan dasar tersebut menjadi layaknya aplikasi siap guna.

13. Component adalah element yang terdiri dari beberapa properti yang dibuat menjadi satu group.

    > fungsinya untuk mempercepat pengembangan dan menyeragamkan design 
    > penyimpanan berada pada assets
    > contoh membuat component button
	- klik kanan componen button (pilih create component)
	- maka component button dibuat di dan tersimpan di assets
	- penggunakan ulang tinggal drag ke frame target
